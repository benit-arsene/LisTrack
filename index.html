Automatic Tracking

Tracks active time per site. Pauses when you're idle or switch tabs.

Beautiful Dashboard

View your time by day, week, month, or the last 7 days. Dark mode included.

Daily Goals

Set screen-time budgets for specific sites. Get notified when you're close to the limit.

Installation Guide

Follow these steps to install LisTrack in Chrome.

1

Download the extension

Click the download button above to get lisTrack-extension.zip.

2

Unzip the file

Extract the contents to a folder on your computer. Make sure you remember where you saved it — you'll need it in the next step.

3

Open Chrome Extensions page

Open a new tab and go to chrome://extensions/ — or click ⋮ Menu → Extensions → Manage Extensions.

4

Enable Developer Mode

Toggle on Developer mode in the top-right corner of the extensions page.

5

Load the extension

Click Load unpacked and select the folder you extracted in step 2. The LisTrack icon will appear in your toolbar.

6

Open the dashboard

Click the LisTrack icon in your toolbar, then click Open Dashboard to view your screen time data. Start browsing — the tracker works automatically!
